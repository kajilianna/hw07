# 肺炎X光片二分类实验报告

## 1. 数据集统计
本次实验使用Kaggle Chest X-Ray Pneumonia数据集，按8:2拆分训练集与验证集，测试集为原始test目录。
| 数据集 | 样本总数 | Normal（正常） | Pneumonia（肺炎） | 类别不平衡比 |
|--------|----------|----------------|-------------------|--------------|
| Train  | 4176     | 1144           | 3032              | 1:2.65       |
| Val    | 1044     | 286            | 758               | 1:2.65       |
| Test   | 624      | 234            | 390               | 1:1.67       |

## 2. 模型结构与超参数
### 模型结构
使用ResNet50迁移学习模型，冻结预训练底层参数，仅微调顶层全连接层：
- 输入：224×224×3的X光片图像
- 主干：ResNet50预训练特征提取层
- 分类头：全连接层输出二分类结果（正常/肺炎）

### 超参数配置
| 超参数         | 取值       |
|----------------|------------|
| 图像尺寸       | 224×224    |
| Batch Size     | 32         |
| 训练轮数(Epochs) | 10         |
| 优化器         | Adam       |
| 学习率         | 1e-4       |
| 损失函数       | 带权重交叉熵 |

## 4. 结果分析与思考
1.  **指标权衡**：本次实验中模型在测试集上出现了高准确率、高召回率的情况，召回率（0.97）高于精确率（0.91）。从医学诊断角度，召回率比准确率更重要，因为肺炎诊断中，漏诊（假阴性）会延误患者治疗，造成严重后果，而误诊（假阳性）可通过进一步检查排除。因此优先保证高召回率是合理的。

2.  **数据增强与迁移学习的作用**：数据增强（随机翻转、旋转）有效缓解了过拟合，提升了模型泛化能力；迁移学习使用ImageNet预训练的ResNet50，利用通用图像特征大幅减少了训练难度，在少量数据下快速收敛，比从零训练的CNN效果提升明显。

3.  **误判后果分析**：将肺炎误判为正常（假阴性）会导致患者错过治疗时机，加重病情甚至危及生命；将正常误判为肺炎（假阳性）会增加患者不必要的检查与治疗成本，造成医疗资源浪费，但相比假阴性后果较轻。

## 5. 进阶挑战（可选）：三分类任务
将任务扩展为Normal/Viral/Bacterial三分类，使用ResNet50微调后，测试集准确率为0.88，混淆矩阵显示细菌性肺炎与病毒性肺炎存在一定混淆，分类难度高于二分类，因为不同肺炎的影像特征差异较小，模型需要学习更细粒度的特征。
